## Affine Spaces

### 2.1 仿射子集的有限张成性与结构

**题目摘要**：证明 $\mathbb{R}^n$ 的任何仿射子集 $A$ 都由一个有限子集张成，并推导出非空仿射子集是某个子向量空间的平移。

**解答**：

1. **有限张成性**：在 $\mathbb{R}^n$ 中，任何仿射独立集的元素个数最多为 $n+1$。因此，我们可以从 $A$ 中选择一个元素个数最大的仿射独立子集 $S = \{p_0, p_1, \dots, p_m\}$，其中 $m \le n$。
2. **极大性证明**：设 $p_{m+1}$ 是 $A$ 中不在 $S$ 里的任意一点。根据 $S$ 的极大性，$S \cup {p_{m+1}}$ 必然不是仿射独立的。这意味着存在不全为零的实数 $s_i$ 满足 $\sum_{i=0}^{m+1} s_i p_i = 0$ 且 $\sum_{i=0}^{m+1} s_i = 0$。
3. **系数推导**：由于 $S$ 本身是仿射独立的，系数 $s_{m+1}$ 绝不能为 0（否则 $s_0, \dots, s_m$ 必须全为 0，矛盾）。由此可得： 
4. $$p_{m+1} = \sum_{i=0}^{m} \left( \frac{-s_i}{s_{m+1}} \right) p_i$$ 由于 $\sum_{i=0}^m \frac{-s_i}{s_{m+1}} = \frac{1}{s_{m+1}} (- \sum_{i=0}^m s_i) = \frac{s_{m+1}}{s_{m+1}} = 1$，说明 $p_{m+1}$ 是 $S$ 的仿射组合。因此，$A$ 由有限集 $S$ 张成。
5. **结论**：根据 **Corollary 2.6**，若 $A$ 是由仿射独立集 ${p_0, \dots, p_m}$ 张成的仿射集，则 $A$ 必然是 $\mathbb{R}^n$ 中某个 $m$ 维子向量空间 $V$ 的平移，即 $A = V + p_0$。

### 2.2 仿射张成与环境空间的不变性


**题目摘要**：证明若 $X \subseteq \mathbb{R}^n$ 且 $\mathbb{R}^n$ 同构于 $\mathbb{R}^k$ 的一个子空间（$n < k$），则 $X$ 在 $\mathbb{R}^n$ 中张成的仿射集与在 $\mathbb{R}^k$ 中张成的仿射集相同。
在该命题中，“$X$ 在 $\mathbb{R}^k$ 中张成的仿射集”应当理解为 **“$\phi(X)$ 在 $\mathbb{R}^k$ 中张成的仿射集”**，其中 $\phi$ 是将 $\mathbb{R}^n$ 映入 $\mathbb{R}^k$ 某个子空间的同构映射。
**解答**：

1. **映射定义**：设 $\phi$ 是向量空间同构。对于 $\mathbb{R}^n$ 中的任何仿射集 $A$，由于 $\phi$ 保持线性组合和仿射组合，其像 $\phi(A)$ 在 $\mathbb{R}^k$ 中依然是仿射的。
2. **张成集包含关系**：
    - 设 $Aff_n(X)$ 是 $\mathbb{R}^n$ 中包含 $X$ 的所有仿射集的交集。其在 $\mathbb{R}^k$ 中的像 $\phi(Aff_n(X))$ 是包含 $\phi(X)$ 的一个仿射集。
    - 反之，由于 $\phi$ 是同构，任何在 $\mathbb{R}^k$ 中包含 $\phi(X)$ 且位于 $\phi(\mathbb{R}^n)$ 范围内的仿射集，其原像在 $\mathbb{R}^n$ 中也是包含 $X$ 的仿射集。
3. **结论**：仿射张成的定义不依赖于外围空间的大小，仅取决于点集本身的仿射组合性质。因此，$X$ 在两个空间中张成的仿射集是同一几何对象的不同表现形式。

## Affine Maps
### 2.3 球面 $S^n$ 上的仿射独立点集

**题目摘要**：证明 $n$ 维球面 $S^n$ 中包含一个拥有 $n+2$ 个点的仿射独立集。

**解答**：

1. **环境观察**：球面 $S^n$ 自然地嵌入在 $\mathbb{R}^{n+1}$ 中。根据定义，在 $\mathbb{R}^{n+1}$ 空间内，一个仿射独立集最多可以包含 $(n+1) + 1 = n+2$ 个点。
2. **利用一般位置定理**：根据 **Theorem 2.7**，对于任何 $k \ge 0$，欧氏空间 $\mathbb{R}^{n+1}$ 都包含 $k$ 个处于一般位置的点。所谓一般位置，意味着这组点中的任何 $n+2$ 个点都是仿射独立的。
3. **归纳构造法**：
    - 对于 $S^0 = {1, -1}$，这两个点在 $\mathbb{R}^1$ 中是仿射独立的（$n+2 = 2$）。
    - 通过归纳，假设 $S^{n-1}$ 在 $\mathbb{R}^n$ 中有 $n+1$ 个仿射独立点。
    - 我们将 $S^{n-1}$ 视为 $S^n$ 的赤道（即最后一维坐标为 0 的点集），并在 $S^n$ 上额外选取北极点 $p_{n+1} = (0, \dots, 0, 1)$。
    - 由于北极点的最后一维坐标不为 0，而赤道点集张成的仿射超平面（最后一维为 0）并不包含它，根据仿射独立性的定义，这 $n+2$ 个点在 $\mathbb{R}^{n+1}$ 中是仿射独立的。



### 2.4 仿射映射的线性分解定理

**题目摘要**：证明若 $T: \mathbb{R}^n \to \mathbb{R}^k$ 是仿射映射，则它可以分解为一个线性变换 $A$ 与一个常数向量 $Y_0$ 的和，即 $T(x) = A(x) + Y_0$。

**解答**：

1. **定义平移项**：令 $Y_0 = T(0)$。
2. **构造线性候选映射**：定义 $A(x) = T(x) - Y_0$。我们需要证明 $A$ 是一个线性变换。
3. **验证齐次性**：对于任意 $x \in \mathbb{R}^n$ 和 $c \in \mathbb{R}$，利用仿射组合的性质（系数和为 1）： $$A(cx) = T(cx + (1-c)0) - T(0) = cT(x) + (1-c)T(0) - T(0) = c(T(x) - T(0)) = cA(x)$$
4. **验证加法性**：对于 $x, y \in \mathbb{R}^n$，注意 $x+y$ 可写作 $2(\frac{1}{2}x + \frac{1}{2}y)$，利用仿射性质可证 $A(x+y) = A(x) + A(y)$。
5. **结论**：由于 $A$ 满足线性变换的定义，且 $T(x) = A(x) + T(0)$，定理得证。

### 2.5 仿射映射的连续性

**题目摘要**：证明每一个仿射映射都是连续的。

**解答**：

1. **利用结构分解**：根据习题 2.4，任何仿射映射 $T$ 都可以写成 $T(x) = A(x) + Y_0$，其中 $A$ 是线性变换。
2. **线性映射的连续性**：在有限维欧氏空间之间，每一个线性变换都是连续的。
3. **平移的连续性**：常数平移映射 $x \mapsto Y_0$ 显然是连续的。
4. **结论**：连续函数的和仍为连续函数，因此 $T$ 必然连续。

### 2.6 $m$-单形的仿射同胚性

**题目摘要**：证明任意两个 $m$-单形之间都存在一个仿射映射构成的同胚。

**解答**：

1. **映射定义**：设两个 $m$-单形分别为 $S = [p_0, \dots, p_m]$ 和 $S' = [q_0, \dots, q_m]$。根据定理 2.10，存在唯一的仿射映射 $T: S \to S'$ 满足 $T(p_i) = q_i$。
2. **双射性证明**：由于 ${p_i}$ 和 ${q_i}$ 均是仿射独立的，单形内的点由重心坐标唯一确定。$T$ 将 $S$ 中的重心组合 $\sum t_i p_i$ 唯一地映射到 $S'$ 中具有相同系数的重心组合 $\sum t_i q_i$，因此 $T$ 是双射。
3. **同胚性结论**：$T$ 是仿射映射故连续（习题 2.5）。其逆映射 $T^{-1}$ 也是由 $T^{-1}(q_i) = p_i$ 定义的仿射映射，同样连续。
4. **结论**：$T$ 是连接任意两个 $m$-单形的仿射同胚。

### 2.7 一维仿射映射的显式公式

**题目摘要**：给出将区间 $[s_1, s_2]$ 映射到 $[t_1, t_2]$ 且满足 $\theta(s_i) = t_i$ 的仿射映射公式，并特化到华氏度与摄氏度的转换公式。

**解答**：

1. **通用公式推导**：设 $\theta(x) = ax + b$。由已知条件：
    - $as_1 + b = t_1$
    - $as_2 + b = t_2$ 解得斜率 $a = \frac{t_2 - t_1}{s_2 - s_1}$，进而推导出通用点斜式公式： 
    - $$\theta(x) = \frac{t_2 - t_1}{s_2 - s_1}(x - s_1) + t_1 \quad$$
2. **特定区间计算**：针对  （华氏度水的冰点与沸点）映射到 （摄氏度）：
    - $a = \frac{100 - 0}{212 - 32} = \frac{100}{180} = \frac{5}{9}$
    - $t_1 = 0, s_1 = 32$
3. **最终公式**： 
$$\theta(x) = \frac{5}{9}(x - 32) \quad$$


### 2.8 仿射映射对仿射集与凸集的保持性

**题目摘要**：证明仿射映射 $T$ 保持子集的仿射性与凸性，并说明其对线段映射的效果。

**解答**：

1. **保持性证明**：设 $A \subseteq \mathbb{R}^n$ 是仿射集，$T: A \to \mathbb{R}^k$ 是仿射映射。设 $X \subseteq A$ 是仿射集（或凸集）。对于任意 $T(x), T(x') \in T(X)$，考虑其线性组合 $tT(x) + (1-t)T(x')$。由于 $T$ 是仿射映射，该组合等于 $T(tx + (1-t)x')$。若 $X$ 是仿射集，$tx + (1-t)x' \in X$ 对所有 $t \in \mathbb{R}$ 成立；若 $X$ 是凸集，则对所有 $t \in$ 成立。因此，其像点必在 $T(X)$ 中，证得 **$T(X)$ 仍为仿射集（或凸集）**。
2. **线段映射**：线段 $t$ 是端点 $a, b$ 的凸组合集合。根据上述性质，$T(t)$ 是 $T(a)$ 与 $T(b)$ 的所有凸组合。
    - 若 $T(a) \neq T(b)$，则 $T(t)$ 是连接这两点的 **线段**。
    - 若 $T(a) = T(b)$，则所有像点重合，$T(t)$ **塌缩为单点** $T(a)$。

### 2.9 包含重心点的仿射独立性

**题目摘要**：证明若 ${p_0, \dots, p_m}$ 仿射独立，则将其中的 $p_i$ 替换为重心 $b$ 后得到的集合仍仿射独立。

**解答**：

1. **假设条件**：设 $b = \frac{1}{m+1} \sum_{j=0}^m p_j$ 是该单形的重心。不失一般性，设被删除的顶点为 $p_0$。
2. **建立方程**：设一组实数 $s_i$ 和 $s$ 满足 $s b + \sum_{i=1}^m s_i p_i = 0$ 且 $s + \sum_{i=1}^m s_i = 0$。
3. **代换分析**：将 $b$ 的定义代入得：$\frac{s}{m+1} p_0 + \sum_{i=1}^m (s_i + \frac{s}{m+1}) p_i = 0$。
4. **系数推导**：由于 ${p_0, \dots, p_m}$ 仿射独立，所有系数必为 0。
    - 由 $p_0$ 的系数得 $s = 0$。
    - 代入 $s = 0$ 到其他项系数得 $s_i = 0$（$i=1, \dots, m$）。
5. **结论**：所有系数均为 0，故 **${b, p_1, \dots, p_m}$ 仿射独立**。

### 2.10 单形与锥空间的同胚性

**题目摘要**：证明 $m$-单形 $[p_0, \dots, p_m]$ 同胚于以 $p_i$ 为顶点、其余点生成的面为底面的锥空间。


#### 1. 定义相关空间

- **$m$-单形 $S$**：由仿射独立集 ${p_0, \dots, p_m}$ 张成的凸集。单形内的任一点 $x$ 都可以唯一地表示为重心组合：$x = \sum_{j=0}^m t_j p_j$，其中 $\sum_{j=0}^m t_j = 1$ 且 $t_j \ge 0$,。
- **底面 $F$**：即单形中与顶点 $p_i$ 相对的 $(m-1)$-维面 $F = [p_0, \dots, \hat{p_i}, \dots, p_m]$。
- **锥空间 $CF$**：定义为商空间 $(F \times I) / (F \times {1})$，其中 $F \times {1}$ 被坍缩为一个点，称为顶点 $v$。

#### 2. 构造映射 $h: S \to CF$

为了证明同胚，我们利用重心坐标构造一个显式映射。不失一般性，设 $i=0$。 对于 $S$ 中的点 $x = t_0 p_0 + \sum_{j=1}^m t_j p_j$：

1. 如果 **$t_0 = 1$**，则 $x = p_0$。将其映至锥空间的顶点：$h(p_0) = v = [y, 1]$（其中 $y \in F$ 任意）。
2. 如果 **$t_0 < 1$**，令底面上的对应点为 $x' = \sum_{j=1}^m \frac{t_j}{1-t_0} p_j$。由于 $\sum_{j=1}^m \frac{t_j}{1-t_0} = \frac{1-t_0}{1-t_0} = 1$ 且系数非负，故 $x' \in F$。 定义映射：**$h(x) = [x', t_0] \in CF$**。

#### 3. 构造逆映射 $k: CF \to S$

定义映射 $k: F \times I \to S$ 为： 
$$k(y, t) = t p_0 + (1-t) y$$

- **良定义性**：当 $t=1$ 时，$k(y, 1) = p_0$，这与 $y$ 的选择无关。因此，$k$ 在商空间 $CF$ 上诱导了一个良定义的映射 $k: CF \to S$。
- **连续性**：由于重心坐标 $t_j$ 是点 $x$ 的连续函数，$h$ 在 $S \setminus {p_0}$ 上显然连续；而在 $p_0$ 处，当 $t_0 \to 1$ 时，像点趋于顶点 $v$。对于逆映射 $k$，由于它是由 $F \times I$ 上的连续映射通过商映射诱导的，且在被标识的纤维上为常数，因此 $k$ 也是连续的。

#### 4. 验证同胚性

- **双射性**：由于重心坐标的唯一性，对于给定的高度 $t$ 和底面点 $x'$，在单形 $S$ 中有且仅有一个点 $x$ 与之对应。
- **结论**：由于 $S$ 是紧致空间且 $CF$ 是豪斯多夫空间，连续双射 $h$ 必然是同胚映射,。

**直观理解**： 这一构造本质上是将 $m$-单形看作是由顶点 $p_i$ 向底面 $F$ 发出的所有线段的并集。重心坐标 $t_i$ 恰好代表了点在“高度”方向上的位置（$t_i=1$ 在顶点，$t_i=0$ 在底面），这与锥空间的几何定义完全吻合。
### 2.11 $n$-单形到 $n$ 维圆盘 $D^n$ 的显式同胚

**题目摘要**：给出从 $n$-单形 $[p_0, \dots, p_n]$ 到 $D^n$ 的显式同胚。

**解答**：

1. **单形标准化**：根据习题 2.6，任意 $n$-单形 $[p_0, \dots, p_n]$ 都仿射同胚于标准 $n$-单形 $\Delta^n$。
2. **平移与居中**：将 $\Delta^n$ 通过平移使其重心（或某个内部点）位于原点。
3. **径向伸缩（Radial Stretching）**：定义映射 $h: \Delta^n \to D^n$。对于不为原点的点 $x$，令 $h(x) = \frac{|x|}{|x_{\partial}|} x$，其中 $x_{\partial}$ 是从原点出发通过 $x$ 的射线与单形边界的交点。
4. **结论**：由于 $n$-单形是紧致凸集且包含内点，这种径向伸缩保证了边界点映至球面 $S^{n-1}$，内部点映至 $D^n$ 内部。因此，**$n$-单形与 $D^n$ 同胚**。
