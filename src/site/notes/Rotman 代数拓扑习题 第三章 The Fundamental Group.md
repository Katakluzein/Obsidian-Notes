## The Fundamental Groupoid

### 3.1 定理 1.3 的推广（相对同伦的复合性质）

**题目摘要**：证明在相对同伦的条件下，映射的复合仍然保持同伦性。

**解答**：

1. **定义已知量**：令 $F: X \times I \to Y$ 为 $f_0 \simeq f_1 \text{ rel } A$ 的同伦，$G: Y \times I \to Z$ 为 $g_0 \simeq g_1 \text{ rel } B$ 的同伦。
2. **构造复合同伦**：定义映射 $H: X \times I \to Z$ 如下： $$H(x, t) = \begin{cases} g_0(F(x, 2t)) & 0 \le t \le 1/2 \ G(f_1(x), 2t-1) & 1/2 \le t \le 1 \end{cases}$$
3. **验证连续性**：在 $t = 1/2$ 处，$g_0(F(x, 1)) = g_0(f_1(x))$，而 $G(f_1(x), 0) = g_0(f_1(x))$。由于两部分在交界处相等，由**粘贴引理**可知 $H$ 连续。
4. **验证相对性（rel A）**：设 $a \in A, t \in I$。
    - 若 $t \le 1/2$，$H(a, t) = g_0(F(a, 2t)) = g_0(f_0(a))$（因为 $F$ 保持 $A$ 不动）。
    - 若 $t \ge 1/2$，$H(a, t) = G(f_1(a), 2t-1)$。由于 $f_1(A) \subseteq B$ 且 $G$ 保持 $B$ 不动，有 $H(a, t) = g_0(f_1(a))$。
    - 由于 $f_0|_A = f_1|_A$，上述两式相等，且始终等于 $g_0 f_0(a) = g_1 f_1(a)$。因此 $H$ 是 **$\text{rel } A$** 的同伦。

### 3.2 道路与圆周映射的相对同伦变换

**题目摘要**：(i) 证明若两条道路 $f, g$ 相对端点同伦，则其诱导的圆周映射 $f', g'$ 相对基点 $\{1\}$ 同伦。(ii) 证明该性质对道路连接（`*`运算）也成立。

**解答**：

1. **部分 (i) 构造**：设 $H: f \simeq g \text{ rel } \dot{I}$ 为道路间的同伦。定义 $H'(e^{2\pi i t}, s) = H(t, s)$。这里隐式的定义了从S1到单位区间的同胚，$u$ 是 $I/\{0, 1\} \to S^1$ 的同胚。
    - 由于 $f(0)=f(1)$，映射 $f'$ 在圆周基点处良定义。
    - 因为 $H$ 保持端点固定（即 $H(0, s) = H(1, s) = x_0$），所以 $H'(1, s) = x_0$ 对所有 $s \in I$ 成立。这意味着在同伦过程中，圆周的基点 $1$ 始终被映向 $x_0$，因此 **$f' \simeq g' \text{ rel } \{1\}$**。
2. **部分 (ii) 运算**：若 $f \simeq f_1 \text{ rel } \dot{I}$ 且 $g \simeq g_1 \text{ rel } \dot{I}$，则道路连接满足 $f * g \simeq f_1 * g_1 \text{ rel } \dot{I}$。根据部分 (i) 的结论，将其映至圆周后有 $(f * g)' \simeq (f_1 * g_1)' \text{ rel } \{1\}$。利用恒等式 $(f * g)' = f' * g'$，证得 **$f' * g' \simeq f_1' * g_1' \text{ rel } \{1\}$**。
### 3.3 圆周映射相对同伦与自由同伦的等价性

**题目摘要**：若 $g$ 是常数道路，证明圆周映射 $f' \simeq g'$ 相对基点 $\{1\}$ 同伦的充要条件是它们自由同伦。

**证明**：

1. **必要性 ($\implies$)**：若 $f' \simeq g' \text{ rel } \{1\}$，根据定义，相对同伦必然是一种特殊的自由同伦，结论显然成立。
2. **充分性 ($\impliedby$)**：假设存在自由同伦 $f' \simeq g'$。
    - 因为 $g$ 是常数道路，故 $g': S^1 \to X$ 是常数映射。这意味着 $f'$ 是 **零伦映射（Nullhomotopic）**。
    - 根据 **定理 1.6 (iii)**，如果映射 $f': S^n \to Y$ 零伦，则对于圆周上的任一点 $x_0$（此处为 $1$），存在一个同伦将其映向常数映射且在该点处保持固定。
    - 因此，自由同伦 $f' \simeq g'$ 蕴含了存在保持基点 $\{1\}$ 不动的同伦，即 **$f' \simeq g' \text{ rel } \{1\}$**。


### 3.4 单形边界与正方形边界的零伦性

**题目摘要**：(i)-(ii) 证明 $2$-单形 $\Delta^2$ 边界路径构成的回路在 $X$ 中是零伦的。(iii) 证明正方形边界的四条路径满足 $\alpha \simeq \gamma * \beta * \delta^{-1}$。

**证明**：

1. **部分 (i) 与 (ii)**：
    
    - **几何基础**：$2$-单形 $\Delta^2 = [e_0, e_1, e_2]$ 与二维圆盘 $D^2$ 同胚。
    - **映射构造**：已知 $u: \Delta^2 \to X$ 是连续映射。题目定义的 $u_0, u_1, u_2$ 分别是 $u$ 在单形三条边上的限制映射。路径 $(u_0 * u_1^{-1}) * u_2$ 是从顶点 $e_1$ 出发，经过 $u_0$ 到 $e_2$，再沿 $u_1$ 的逆向回到 $e_0$，最后经 $u_2$ 回到 $e_1$ 的闭回路。
    - **利用定理 1.6**：该定理指出，一个映射 $f: S^1 \to X$ 是零伦的，当且仅当它可以扩张到 $D^2$ 上。由于映射 $u$ 已经定义在整个 $\Delta^2$（即 $D^2$）上，且其在边界上的限制恰好就是该回路，因此该回路在 $X$ 中 **$\text{rel } \dot{I}$ 零伦**。同理可证 (ii) 中的回路也是零伦的。
    - 也可以用简单的几何学来直接构造这个定端同伦。
2. **部分 (iii)**：
    
    - **同伦构造**：设 $F: I \times I \to X$。其四个边界路径定义为：底边 $\alpha(t)=F(t,0)$，顶边 $\beta(t)=F(t,1)$，左边 $\gamma(t)=F(0,t)$，右边 $\delta(t)=F(1,t)$。
    - **显式公式**：构造同伦 $H(s, t): I \times I \to X$，其几何直觉是将底边 $\alpha$ 逐步向上推，经过侧边 $\gamma$ 和 $\delta^{-1}$ 最终贴合到 $\beta$。参考资料给出的分段公式如下： $$H(s, t) = \begin{cases} F(0, 4ts) & 0 \le s \le 1/4 \\ F(4s-1, t) & 1/4 \le s \le 1/2 \\ F(1, 2t(1-s)) & 1/2 \le s \le 1 \end{cases}$$
    - 在 $t=0$ 时，$H(s, 0)$ 缩减为 $\alpha(s)$；在 $t=1$ 时，$H(s, 1)$ 描述了路径连接 $\gamma * \beta * \delta^{-1}$。通过粘贴引理可知该映射连续，故 **$\alpha \simeq \gamma * \beta * \delta^{-1} \text{ rel } \dot{I}$**。

### 3.5 乘积空间中的同伦保持性

**题目摘要**：证明若分量路径分别同伦，则其在乘积空间 $X \times Y$ 中构成的路径也同伦。

**证明**：

1. **前提条件**：已知在 $X$ 中 $f_0 \simeq f_1 \text{ rel } \dot{I}$（同伦为 $F$），在 $Y$ 中 $g_0 \simeq g_1 \text{ rel } \dot{I}$（同伦为 $G$）。
2. **构造整体同伦**：定义 $H: I \times I \to X \times Y$ 为： $$H(s, t) = (F(s, t), G(s, t))$$
3. **验证性质**：
    - **连续性**：映射到乘积空间的函数连续，当且仅当其分量映射（$F$ 和 $G$）均连续。
    - **端点固定**：因为 $F$ 保持端点 $f_i(0), f_i(1)$ 固定，$G$ 保持 $g_i(0), g_i(1)$ 固定，所以 $H(0, t) = (f_0(0), g_0(0))$ 且 $H(1, t) = (f_0(1), g_0(1))$ 对所有 $t$ 恒定。
4. **结论**：$H$ 是连接 $(f_0, g_0)$ 与 $(f_1, g_1)$ 的 **$\text{rel } \dot{I}$ 同伦**。

### 3.6 逆路径与连接运算的代数性质

**题目摘要**：(i) 逆映射保持同伦性；(ii) 连接运算的逆等于逆的连接（次序颠倒）；(iii) 给出 $f * f^{-1} \neq f^{-1} * f$ 的例子；(iv) 证明恒等路径与非恒等路径连接不等于原路径。

**证明**：

1. **部分 (i)**：若 $f \simeq g \text{ rel } \dot{I}$（同伦为 $F(s, t)$），定义逆路径同伦为 $G(s, t) = F(1-s, t)$。由于 $1-s$ 连续且保持端点标识，故 **$f^{-1} \simeq g^{-1} \text{ rel } \dot{I}$**。
2. **部分 (ii)**：根据连接运算 $*$ 的定义： $$(f * g)^{-1}(s) = (f * g)(1-s) = \begin{cases} g(2(1-s)-1) = g(1-2s) & 0 \le s \le 1/2 \ f(2(1-s)) = f(2-2s) & 1/2 \le s \le 1 \end{cases}$$ 这正好符合 **$g^{-1} * f^{-1}$** 的定义（先走 $g$ 的逆，再走 $f$ 的逆）。
3. **部分 (iii)**：取 $S^1$ 上的闭回路 $f(t) = e^{2\pi i t}$。在 $t=1/8$ 处：
    - $(f * f^{-1})(1/8) = f(1/4) = i$。
    - $(f^{-1} * f)(1/8) = f^{-1}(1/4) = f(3/4) = -i$。 因为在某些点处函数值不同，故作为映射 **$f * f^{-1} \neq f^{-1} * f$**。正着走和反着走本来就不一样。
4. **部分 (iv)**：设 $f$ 不是常数路径且 $f(0)=p$。
    - $(i_p * f)(s)$ 在 $s \in [0, 1/2]$ 时恒等于 $p$。
    - 若 $i_p * f = f$，则意味着 $f$ 在 $[0, 1/2]$ 上必须恒等于 $p$。
    - 通过递归（如资料 所示），可推导出 $f$ 在 $[0, 1-2^{-n}]$ 上均恒等于 $p$。当 $n \to \infty$ 时，$f$ 必须是常数路径，这与假设矛盾。
    - 结论：作为映射 **$i_p * f \neq f$**。
