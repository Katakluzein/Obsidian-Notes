## The Fundamental Groupoid

### 3.1 定理 1.3 的推广（相对同伦的复合性质）

**题目摘要**：证明在相对同伦的条件下，映射的复合仍然保持同伦性。

**解答**：

1. **定义已知量**：令 $F: X \times I \to Y$ 为 $f_0 \simeq f_1 \text{ rel } A$ 的同伦，$G: Y \times I \to Z$ 为 $g_0 \simeq g_1 \text{ rel } B$ 的同伦。
2. **构造复合同伦**：定义映射 $H: X \times I \to Z$ 如下： $$H(x, t) = \begin{cases} g_0(F(x, 2t)) & 0 \le t \le 1/2 \ G(f_1(x), 2t-1) & 1/2 \le t \le 1 \end{cases}$$
3. **验证连续性**：在 $t = 1/2$ 处，$g_0(F(x, 1)) = g_0(f_1(x))$，而 $G(f_1(x), 0) = g_0(f_1(x))$。由于两部分在交界处相等，由**粘贴引理**可知 $H$ 连续。
4. **验证相对性（rel A）**：设 $a \in A, t \in I$。
    - 若 $t \le 1/2$，$H(a, t) = g_0(F(a, 2t)) = g_0(f_0(a))$（因为 $F$ 保持 $A$ 不动）。
    - 若 $t \ge 1/2$，$H(a, t) = G(f_1(a), 2t-1)$。由于 $f_1(A) \subseteq B$ 且 $G$ 保持 $B$ 不动，有 $H(a, t) = g_0(f_1(a))$。
    - 由于 $f_0|_A = f_1|_A$，上述两式相等，且始终等于 $g_0 f_0(a) = g_1 f_1(a)$。因此 $H$ 是 **$\text{rel } A$** 的同伦。

### 3.2 道路与圆周映射的相对同伦变换

**题目摘要**：(i) 证明若两条道路 $f, g$ 相对端点同伦，则其诱导的圆周映射 $f', g'$ 相对基点 $\{1\}$ 同伦。(ii) 证明该性质对道路连接（`*`运算）也成立。

**解答**：

1. **部分 (i) 构造**：设 $H: f \simeq g \text{ rel } \dot{I}$ 为道路间的同伦。定义 $H'(e^{2\pi i t}, s) = H(t, s)$。这里隐式的定义了从S1到单位区间的同胚，$u$ 是 $I/\{0, 1\} \to S^1$ 的同胚。
    - 由于 $f(0)=f(1)$，映射 $f'$ 在圆周基点处良定义。
    - 因为 $H$ 保持端点固定（即 $H(0, s) = H(1, s) = x_0$），所以 $H'(1, s) = x_0$ 对所有 $s \in I$ 成立。这意味着在同伦过程中，圆周的基点 $1$ 始终被映向 $x_0$，因此 **$f' \simeq g' \text{ rel } \{1\}$**。
2. **部分 (ii) 运算**：若 $f \simeq f_1 \text{ rel } \dot{I}$ 且 $g \simeq g_1 \text{ rel } \dot{I}$，则道路连接满足 $f * g \simeq f_1 * g_1 \text{ rel } \dot{I}$。根据部分 (i) 的结论，将其映至圆周后有 $(f * g)' \simeq (f_1 * g_1)' \text{ rel } \{1\}$。利用恒等式 $(f * g)' = f' * g'$，证得 **$f' * g' \simeq f_1' * g_1' \text{ rel } \{1\}$**。
### 3.3 圆周映射相对同伦与自由同伦的等价性

**题目摘要**：若 $g$ 是常数道路，证明圆周映射 $f' \simeq g'$ 相对基点 $\{1\}$ 同伦的充要条件是它们自由同伦。

**证明**：

1. **必要性 ($\implies$)**：若 $f' \simeq g' \text{ rel } \{1\}$，根据定义，相对同伦必然是一种特殊的自由同伦，结论显然成立。
2. **充分性 ($\impliedby$)**：假设存在自由同伦 $f' \simeq g'$。
    - 因为 $g$ 是常数道路，故 $g': S^1 \to X$ 是常数映射。这意味着 $f'$ 是 **零伦映射（Nullhomotopic）**。
    - 根据 **定理 1.6 (iii)**，如果映射 $f': S^n \to Y$ 零伦，则对于圆周上的任一点 $x_0$（此处为 $1$），存在一个同伦将其映向常数映射且在该点处保持固定。
    - 因此，自由同伦 $f' \simeq g'$ 蕴含了存在保持基点 $\{1\}$ 不动的同伦，即 **$f' \simeq g' \text{ rel } \{1\}$**。


### 3.4 单形边界与正方形边界的零伦性

**题目摘要**：(i)-(ii) 证明 $2$-单形 $\Delta^2$ 边界路径构成的回路在 $X$ 中是零伦的。(iii) 证明正方形边界的四条路径满足 $\alpha \simeq \gamma * \beta * \delta^{-1}$。

**证明**：

1. **部分 (i) 与 (ii)**：
    
    - **几何基础**：$2$-单形 $\Delta^2 = [e_0, e_1, e_2]$ 与二维圆盘 $D^2$ 同胚。
    - **映射构造**：已知 $u: \Delta^2 \to X$ 是连续映射。题目定义的 $u_0, u_1, u_2$ 分别是 $u$ 在单形三条边上的限制映射。路径 $(u_0 * u_1^{-1}) * u_2$ 是从顶点 $e_1$ 出发，经过 $u_0$ 到 $e_2$，再沿 $u_1$ 的逆向回到 $e_0$，最后经 $u_2$ 回到 $e_1$ 的闭回路。
    - **利用定理 1.6**：该定理指出，一个映射 $f: S^1 \to X$ 是零伦的，当且仅当它可以扩张到 $D^2$ 上。由于映射 $u$ 已经定义在整个 $\Delta^2$（即 $D^2$）上，且其在边界上的限制恰好就是该回路，因此该回路在 $X$ 中 **$\text{rel } \dot{I}$ 零伦**。同理可证 (ii) 中的回路也是零伦的。
    - 也可以用简单的几何学来直接构造这个定端同伦。
2. **部分 (iii)**：
    
    - **同伦构造**：设 $F: I \times I \to X$。其四个边界路径定义为：底边 $\alpha(t)=F(t,0)$，顶边 $\beta(t)=F(t,1)$，左边 $\gamma(t)=F(0,t)$，右边 $\delta(t)=F(1,t)$。
    - **显式公式**：构造同伦 $H(s, t): I \times I \to X$，其几何直觉是将底边 $\alpha$ 逐步向上推，经过侧边 $\gamma$ 和 $\delta^{-1}$ 最终贴合到 $\beta$。参考资料给出的分段公式如下： $$H(s, t) = \begin{cases} F(0, 4ts) & 0 \le s \le 1/4 \\ F(4s-1, t) & 1/4 \le s \le 1/2 \\ F(1, 2t(1-s)) & 1/2 \le s \le 1 \end{cases}$$
    - 在 $t=0$ 时，$H(s, 0)$ 缩减为 $\alpha(s)$；在 $t=1$ 时，$H(s, 1)$ 描述了路径连接 $\gamma * \beta * \delta^{-1}$。通过粘贴引理可知该映射连续，故 **$\alpha \simeq \gamma * \beta * \delta^{-1} \text{ rel } \dot{I}$**。

### 3.5 乘积空间中的同伦保持性

**题目摘要**：证明若分量路径分别同伦，则其在乘积空间 $X \times Y$ 中构成的路径也同伦。

**证明**：

1. **前提条件**：已知在 $X$ 中 $f_0 \simeq f_1 \text{ rel } \dot{I}$（同伦为 $F$），在 $Y$ 中 $g_0 \simeq g_1 \text{ rel } \dot{I}$（同伦为 $G$）。
2. **构造整体同伦**：定义 $H: I \times I \to X \times Y$ 为： $$H(s, t) = (F(s, t), G(s, t))$$
3. **验证性质**：
    - **连续性**：映射到乘积空间的函数连续，当且仅当其分量映射（$F$ 和 $G$）均连续。
    - **端点固定**：因为 $F$ 保持端点 $f_i(0), f_i(1)$ 固定，$G$ 保持 $g_i(0), g_i(1)$ 固定，所以 $H(0, t) = (f_0(0), g_0(0))$ 且 $H(1, t) = (f_0(1), g_0(1))$ 对所有 $t$ 恒定。
4. **结论**：$H$ 是连接 $(f_0, g_0)$ 与 $(f_1, g_1)$ 的 **$\text{rel } \dot{I}$ 同伦**。

### 3.6 逆路径与连接运算的代数性质

**题目摘要**：(i) 逆映射保持同伦性；(ii) 连接运算的逆等于逆的连接（次序颠倒）；(iii) 给出 $f * f^{-1} \neq f^{-1} * f$ 的例子；(iv) 证明恒等路径与非恒等路径连接不等于原路径。

**证明**：

1. **部分 (i)**：若 $f \simeq g \text{ rel } \dot{I}$（同伦为 $F(s, t)$），定义逆路径同伦为 $G(s, t) = F(1-s, t)$。由于 $1-s$ 连续且保持端点标识，故 **$f^{-1} \simeq g^{-1} \text{ rel } \dot{I}$**。
2. **部分 (ii)**：根据连接运算 $*$ 的定义： $$(f * g)^{-1}(s) = (f * g)(1-s) = \begin{cases} g(2(1-s)-1) = g(1-2s) & 0 \le s \le 1/2 \ f(2(1-s)) = f(2-2s) & 1/2 \le s \le 1 \end{cases}$$ 这正好符合 **$g^{-1} * f^{-1}$** 的定义（先走 $g$ 的逆，再走 $f$ 的逆）。
3. **部分 (iii)**：取 $S^1$ 上的闭回路 $f(t) = e^{2\pi i t}$。在 $t=1/8$ 处：
    - $(f * f^{-1})(1/8) = f(1/4) = i$。
    - $(f^{-1} * f)(1/8) = f^{-1}(1/4) = f(3/4) = -i$。 因为在某些点处函数值不同，故作为映射 **$f * f^{-1} \neq f^{-1} * f$**。正着走和反着走本来就不一样。
4. **部分 (iv)**：设 $f$ 不是常数路径且 $f(0)=p$。
    - $(i_p * f)(s)$ 在 $s \in [0, 1/2]$ 时恒等于 $p$。
    - 若 $i_p * f = f$，则意味着 $f$ 在 $[0, 1/2]$ 上必须恒等于 $p$。
    - 通过递归（如资料 所示），可推导出 $f$ 在 $[0, 1-2^{-n}]$ 上均恒等于 $p$。当 $n \to \infty$ 时，$f$ 必须是常数路径，这与假设矛盾。
    - 结论：作为映射 **$i_p * f \neq f$**。

### 3.7 $\sin(1/x)$ 空间基本群平凡性的证明
要证明 $\sin(1/x)$ 空间 $X$ 在任何基点 $x_0$ 处的基本群 $\pi_1(X, x_0)$ 都是平凡群（即 $\{1\}$），我们可以结合其**路连通性**和**路分支的可缩性**进行论证：

**证明步骤：**

1. **定义空间结构**： $\sin(1/x)$ 空间定义为 $X = A \cup G \subset \mathbb{R}^2$：
    
    - $A = {(0, y) : -1 \le y \le 1}$ 是 $y$ 轴上的闭线段。
    - $G = {(x, \sin(1/x)) : 0 < x \le 1/2\pi}$ 是连续函数的图像。
2. **识别路分支 (Path Components)**： 虽然 $X$ 是连通的（因为 $A$ 包含在 $G$ 的闭包内），但它**不是道路连通的**。该空间恰好有两个路分支：
    
    - **路分支 $A$**：是一个闭线段。
    - **路分支 $G$**：是连续函数在区间上的图像，同胚于半开半闭区间。
3. **路分支的可缩性 (Contractibility)**：
    
    - $A$ 作为欧氏空间中的凸集（线段），是**可缩的**。
    - $G$ 同胚于区间，因此也是**可缩的**。
4. **利用基本群的性质**： 根据定理 3.5，设 $x_0$ 是空间 $X$ 中的一点，$X_0$ 是包含 $x_0$ 的路分支，则包含映射诱导了一个同构： $$\pi_1(X_0, x_0) \cong \pi_1(X, x_0)$$ 这意味着一个空间的基本群完全由包含基点的那个路分支决定。
    
5. **得出结论**：
    
    - 由于 $X$ 的每一个路分支（$A$ 或 $G$）都是可缩空间。
    - 根据推论 3.12，**任何可缩空间的基本群都是平凡的**（即 $\pi_1 = \{1\}$）。
    - 因此，无论基点 $x_0$ 选在 $A$ 中还是 $G$ 中，相应的路分支基本群都为 $\{1\}$。

**最终结论：** 对于每一个 $x_0 \in X$，$\pi_1(X, x_0) = \{1\}$。


### 3.8 非局部道路连通的可缩空间

**题目摘要**：给出一个可缩（Contractible）但非局部道路连通（Not locally path connected）的拓扑空间示例。

**解答**：

1. **构造示例**：设 $X$ 为 **$\sin(1/x)$ 空间**。考虑其**锥空间 (Cone)** $CX$。
2. **证明可缩性**：根据 Rotman 书中的 **Theorem 1.11**，对于任何拓扑空间 $X$，其对应的锥空间 $CX$ 都是可缩的。因此 $CX$ 是一个可缩空间。
3. **证明非局部道路连通性**：
    - 考虑 $CX$ 基底上的点 $x = ((0, 0), 0)$，即原 $\sin(1/x)$ 空间在 $y$ 轴线段上的一个点。
    - 取 $x$ 的一个足够小的邻域 $U$（使其不包含锥的顶点 $[x, 1]$）。在 $U$ 中选取属于 $\sin(1/x)$ 曲线上的点 $y = (( \epsilon, \sin(1/\epsilon)), 0)$。
    - 如果 $U$ 内存在连接 $x$ 和 $y$ 的道路，那么通过锥空间的投影映射（它是连续的），该道路将诱导一条在原空间 $X$ 中连接 $(0,0)$ 与 $(\epsilon, \sin(1/\epsilon))$ 的道路。
    - 由于 $\sin(1/x)$ 空间在这些点之间不是道路连通的（Example 1.8），这种道路不存在。
    - 因此，在该点附近的任意小邻域内都无法保证道路连通性，证得 **$CX$ 不是局部道路连通的**。

### 3.9 基本广群范畴的构造与性质

**题目摘要**：证明对于空间 $X$，存在一个范畴 $\mathcal{C}$，其对象为 $X$ 中的点，态射为道路类 $[f]$，并证明该范畴中的每个态射都是等价（Equivalence）。

**证明**：

1. **验证范畴公理**：
    - **对象与态射**：定义对象类 $\text{obj } \mathcal{C} = X$。对于点 $p, q \in X$，态射集 $\text{Hom}(p, q)$ 为所有从 $p$ 到 $q$ 的道路类 $[f]$（即相对端点同伦类）。
    - **复合运算**：定义态射的复合为道路的连接运算 $[f][g] = [f * g]$，其中 $w([f]) = \alpha([g]) = q$。
    - **结合律**：根据 **Theorem 3.2(ii)**，道路类的连接运算在定义良好的情况下满足结合律，即 $([f][g])[h] = [f]([g][h])$。
    - **恒等态射**：对于每个点 $p$，常数道路类 $[i_p]$ 充当恒等元。根据 **Theorem 3.2(i)**，满足 $[i_p][f] = [f]$ 且 $[f][i_q] = [f]$。
2. **证明等价性 (Equivalence)**：
    - 在范畴论中，一个态射 $[f]: p \to q$ 是**等价**的，当且仅当存在反向态射 $[g]: q \to p$ 满足 $[f][g] = 1_p$ 且 $[g][f] = 1_q$。
    - 对于任意道路类 $[f]$，取其**逆道路类 $[f^{-1}]$** 作为候选逆元，其中 $f^{-1}(t) = f(1-t)$。
    - 根据 **Theorem 3.2(iii)**，$[f][f^{-1}] = [i_p]$ 且 $[f^{-1}][f] = [i_q]$。
    - 由于每一个态射都存在这种双侧逆元，因此 **$\mathcal{C}$ 中的每一个态射都是等价态射**。

该范畴在数学上被称为空间 $X$ 的**基本广群 (Fundamental Groupoid)**。

### 3.10 $\pi_0$ 作为从 $\mathsf{Top}_*$ 到 $\mathsf{Sets}_*$ 的函子

**题目摘要**：证明若将包含基点 $x_0$ 的道路连通分支作为 $\pi_0(X)$ 的基点，则 $\pi_0$ 定义了一个从带基点拓扑空间范畴到带基点集合范畴的函子。

**证明**：

1. **对象的映射**： 对于 $\mathsf{Top}_*$ 中的任一对象 $(X, x_0)$，定义 $\pi_0(X, x_0) = (\pi_0(X), [x_0])$，其中 $\pi_0(X)$ 是 $X$ 的所有道路连通分支的集合。基点 $[x_0]$ 是包含 $x_0$ 的那个唯一的道路连通分支。因此，映射后的对象是一个带基点的集合，属于 $\mathsf{Sets}_*$。
2. **态射的映射**： 对于 $\mathsf{Top}_*$ 中的态射（即保持基点的连续映射）$f: (X, x_0) \to (Y, y_0)$，定义 $\pi_0(f): \pi_0(X) \to \pi_0(Y)$ 为将 $X$ 的道路分支 $C$ 映射为包含像集 $f(C)$ 的那个唯一的 $Y$ 的道路分支。由于连续映射将**道路连通集映射为道路连通集**，这一映射是良定义的。
3. **保持基点**： 因为 $f$ 是保持基点的映射，有 $f(x_0) = y_0$。由于 $x_0$ 属于分支 $[x_0]$，那么 $f(x_0) = y_0$ 必然属于分支 $f([x_0])$。根据定义，$\pi_0(f)([x_0])$ 即为包含 $y_0$ 的分支 $[y_0]$。因此，$\pi_0(f)$ 是 $\mathsf{Sets}_*$ 中的态射（保持基点的函数）。
4. **函子公理验证**：
    - **保持恒等映射**：$\pi_0(1_X)$ 将分支 $[x]$ 映至包含 $1_X([x]) = [x]$ 的分支，即自身，故 $\pi_0(1_X) = 1_{\pi_0(X)}$。
    - **保持复合**：对于 $X \xrightarrow{f} Y \xrightarrow{g} Z$，$\pi_0(g \circ f)$ 将分支 $C$ 映至包含 $g(f(C))$ 的分支，这与先应用 $\pi_0(f)$ 再应用 $\pi_0(g)$ 的结果一致，即 $\pi_0(g \circ f) = \pi_0(g) \circ \pi_0(f)$。 综上所述，$\pi_0$ 是一个从 $\mathsf{Top}_*$ 到 $\mathsf{Sets}_*$ 的**协变函子**。

### 3.11 单点空间的基本群

**题目摘要**：证明若 $X = {x_0}$ 是一个单点空间，则其基本群 $\pi_1(X, x_0)$ 是平凡群 $\{1\}$。

**证明**：

1. **路径的唯一性**： 基本群 $\pi_1(X, x_0)$ 的元素是以 $x_0$ 为端点的闭合道路的同伦类 $[f]$。在单点空间 $X = {x_0}$ 中，对于任何连续映射 $f: I \to X$，由于定义域 $I$ 的所有点都必须被映入其唯一的陪域元素中，因此对于所有 $t \in I$，必有 $f(t) = x_0$。
2. **常数道路**： 这意味着在单点空间中，唯一可能的道路是常数道路 $i_{x_0}$。
3. **结论**： 由于只存在唯一的一条道路，因此也只存在唯一的一个道路同伦类 $[i_{x_0}]$。根据群的定义，这个唯一的元素必须是单位元。因此，$\pi_1(X, x_0)$ 是仅包含单位元的**平凡群**，记作 $\{1\}$。

### 3.12 $S^1$ 基本群非平凡性的必要性证明

**题目摘要**：证明若存在一个带基点空间 $(Y, y_0)$ 使得 $\pi_1(Y, y_0) \neq \{1\}$，则 $\pi_1(S^1, 1) \neq \{1\}$。

**证明过程**：

1. **基本群的定义与映射视图**： 根据基本群的范畴论视角，$\pi_1(Y, y_0)$ 可以视为从单位圆周 $(S^1, 1)$ 到带基点空间 $(Y, y_0)$ 的所有**保持基点的同伦类**的集合。即 $\pi_1(Y, y_0) \cong [(S^1, 1), (Y, y_0)]$。在这个语境下，每一个闭道路 $f: I \to Y$ 都对应一个圆周映射 $f': S^1 \to Y$。
    
2. **反证法假设**： 假设 $\pi_1(S^1, 1)$ 是平凡群，即 $\pi_1(S^1, 1) = \{1\}$。
    
3. **推导恒等映射的零伦性**： 在 $\pi_1(S^1, 1)$ 中，恒等映射 $1_{S^1}: (S^1, 1) \to (S^1, 1)$ 代表了该群的一个元素。如果该群是平凡的，那么所有的元素都必须与单位元（即常数映射类）相等。这意味着 $1_{S^1}$ 在保持基点的意义下与常数映射 $c$ 同伦，即 **$1_{S^1} \simeq c \text{ rel } \{1\}$**。
    
4. **利用同伦的复合性质（Theorem 1.3）**： 根据同伦映射的复合性质，如果两个映射 $h$ 和 $k$ 是同伦的（$h \simeq k$），那么对于任何连续映射 $f$，其**复合映射也满足** $f \circ h \simeq f \circ k$。 设 $[f]$ 是 $\pi_1(Y, y_0)$ 中的任意元素，其中 $f: (S^1, 1) \to (Y, y_0)$。我们可以将 $f$ 写为： $$f = f \circ 1_{S^1}$$
    
5. **导出矛盾**： 由于我们假设 $1_{S^1} \simeq c$，通过左复合 $f$，可以得出： $$f = f \circ 1_{S^1} \simeq f \circ c$$。 因为 $c$ 是将 $S^1$ 映向基点 $1$ 的常数映射，所以 $f \circ c$ 是将 $S^1$ 映向 $f(1) = y_0$ 的 **$Y$ 中的常数映射**。 这意味着 $Y$ 中的每一个闭道路 $f$ 都是 **零伦的**（nullhomotopic），即对于每一个 $[f] \in \pi_1(Y, y_0)$，都有 $[f] = [c] = 1$。
    
6. **结论**： 这直接导致 $\pi_1(Y, y_0) = \{1\}$，这与题目中“$\pi_1(Y, y_0) \neq \{1\}$”的已知前提相矛盾。 因此，最初的假设不成立，我们必须有 **$\pi_1(S^1, 1) \neq \{1\}$**。


### 3.13 证明：$[u]$ 是 $\pi_1(S^1, 1)$ 的生成元

**证明背景**： 根据定理 3.16，存在一个群同构映射 $d: \pi_1(S^1, 1) \to \mathbb{Z}$，该映射将每一个闭道路类 $[f]$ 映至其**度（Degree）** $\deg f$。由于 $\mathbb{Z}$ 是由 $1$ 生成的循环群，要证明 $[u]$ 是 $\pi_1(S^1, 1)$ 的生成元，只需证明 $\deg u = 1$。

**证明步骤**：

1. **定义映射**：给定闭道路 $u: I \to S^1$，定义为 $u(t) = \exp(t) = e^{2\pi i t}$。
2. **构造提升（Lifting）**：根据度数的定义，我们需要找到 $u$ 的唯一连续提升 $\tilde{u}: I \to \mathbb{R}$，使得 $\exp \circ \tilde{u} = u$ 且 $\tilde{u}(0) = 0$。
3. **确定提升函数**：令 $\tilde{u}(t) = t$。

    - 因此，$\tilde{u}(t) = t$ 是 $u$ 符合条件的唯一提升。
4. **计算度数**：根据公式 $\deg u = \tilde{u}(1)$，得到 $\deg u = 1$。
5. **结论**：因为 $d: [u] \mapsto 1$ 是同构映射且 $1$ 生成 $\mathbb{Z}$，所以 $[u]$ 是基本群 $\pi_1(S^1, 1)$ 的生成元。(实际上文中给出了d的制作，只需要说这个就行)

### 3.14 证明：$\deg(f^m) = m \cdot \deg f$

**证明背景**： 设 $f$ 是 $S^1$ 在基点 $1$ 处的闭道路。对于整数 $m \in \mathbb{Z}$，定义新道路 $f^m: t \mapsto f(t)^m$。我们需要证明这条道路的度数满足线性倍数关系。

**证明步骤**：

1. **设定已知量**：令 $\tilde{f}: I \to \mathbb{R}$ 为 $f$ 的唯一提升，满足 $\tilde{f}(0) = 0$。根据定义，有 $\deg f = \tilde{f}(1)$。
2. **构造 $f^m$ 的提升**：定义函数 $\tilde{g}: I \to \mathbb{R}$ 为 $\tilde{g}(t) = m \cdot \tilde{f}(t)$。（证明的关键在于制作提升）
3. **验证提升的合法性**：
    - **复合条件**：$\exp \tilde{g}(t) = e^{2\pi i (m \tilde{f}(t))} = (e^{2\pi i \tilde{f}(t)})^m$。
    - 由于 $\exp \tilde{f}(t) = f(t)$，则有 $\exp \tilde{g}(t) = f(t)^m = f^m(t)$。
    - **初值条件**：$\tilde{g}(0) = m \cdot \tilde{f}(0) = m \cdot 0 = 0$。
    - 因此，$\tilde{g}$ 是 $f^m$ 满足起始点为 $0$ 的唯一连续提升。
4. **计算度数**： 根据度数定义： $$\deg(f^m) = \tilde{g}(1) = m \cdot \tilde{f}(1)$$ 代入 $\deg f = \tilde{f}(1)$，得： $$\deg(f^m) = m \cdot \deg f$$
5. **结论**：命题得证。这说明在圆周映射中，将复数坐标提升到 $m$ 次幂，其在基本群中对应的缠绕数（度数）也相应地变为 $m$ 倍。


### 3.15 不同基点的道路同伦的充要条件
**题目摘要**： 设 $f: (I, \dot{I}) \to (S^1, a)$ 是基点为 $a$ 的闭道路。定义 $\text{deg } f = \text{deg } (R \circ f)$，其中 $R: S^1 \to S^1$ 是将 $a$ 映至 $1$ 的旋转映射。证明：两条闭道路 $f$（基点 $a$）与 $g$（基点 $b$）在同伦过程中始终保持为闭道路（即自由同伦）的充要条件是它们具有相同的度（degree）。

**证明过程**：

#### 1. 必要性 ($\implies$)

 $f$ 和 $g$ 同伦，等价于存在圆周映射之间的同伦 $f' \simeq g': S^1 \to S^1$，存在同伦 $H: I \times I \to S^1$，使得对于每个固定的 $t \in I$，道路 $s \mapsto H(s, t)$ 都是闭道路。

- 根据 **引理 3.8** 和 **推论 3.9(ii)**，如果两个映射自由同伦，且目标空间的基本群是阿贝尔群（$\pi_1(S^1) \cong \mathbb{Z}$ 是阿贝尔群），则它们诱导的基本群同态相等。
- 由于基本群同态 $f'_*$ 对应于乘以 $\text{deg } f$ 的运算，因此 $f'_* = g'_*$ 蕴含了 **$\text{deg } f = \text{deg } g$**。


#### 2. 充分性 ($\impliedby$)

假设 $\text{deg } f = \text{deg } g$。

- **利用练习 1.3（旋转映射）**：由于旋转映射 $R_a: a \to 1$ 始终与恒等映射同伦，我们可以将基点 $a$ 和 $b$ 处的道路分别通过旋转“对齐”到基点 $1$ 上。
- 此时，$R_a \circ f$ 和 $R_b \circ g$ 都是基点为 $1$ 的闭道路，且根据定义，它们的度相等。
- **利用推论 3.18**：该推论指出，基点为 $1$ 的两条闭道路相对端点同伦（rel $\dot{I}$）的充要条件是它们度数相等。因此存在相对同伦 $G: (R_a \circ f) \simeq (R_b \circ g) \text{ rel } \dot{I}$。
- **构造整体同伦**：令 $\gamma(t)$ 为连接 $a$ 和 $b$ 的旋转路径， $R_t$ 为将 $1$ 映至 $\gamma(t)$ 的旋转。定义 $H(s, t) = R_t(G(s, t))$。由于 $G$ 在端点处固定为 $1$，则 $H(0, t) = H(1, t) = R_t(1) = \gamma(t)$。
- 这构造了一个从 $f$ 到 $g$ 的同伦，且在每一时刻 $t$ 都是闭道路。

补充说明：
在上述证明中，**$f'$ 是由闭道路 $f$ 诱导的圆周映射（Circular map）**。
1. **构造方式**：设 $f: I \to S^1$ 是 $S^1$ 中的一条闭道路（即 $f(0) = f(1) = a$）。定义 $u: I \to S^1$ 为商映射 $t \mapsto e^{2\pi i t}$，它将区间的端点 ${0, 1}$ 标识为圆周上的基点 $1$。由于 $f$ 在端点处取值相同，根据商拓扑的性质，它可以诱导出一个唯一的连续映射 **$f': S^1 \to S^1$**，使得 $f' \circ u = f$。
2. **数学表达式**：$f'$ 的定义通常记为 **$f'(e^{2\pi i t}) = f(t)$**，其中 $t \in I$。
3. **直观理解**：$f$ 是在时间 $I$ 内在圆周上运动的轨迹，而 $f'$ 则是将这种运动看作圆周本身到圆周的一个连续变换（映射）。
4. **为什么要用 $f'$**：在证明必要性时，将路径间的同伦转化为 $f'$ 与 $g'$ 之间的**自由同伦**，是为了利用圆周自映射在基本群中诱导的同态性质。根据 $S^1$ 的基本群性质，$f'$ 诱导的同态 $f'_*: \mathbb{Z} \to \mathbb{Z}$ 本质上就是**乘以该映射的度数**的操作，即 $f'_*(n) = (\text{deg } f) \cdot n$。



### 3.16：计算环面 $T = S^1 \times S^1$ 的基本群 $\pi_1(T, t_0)$
本证明通过利用**乘积空间的基本群定理**，将环面 $T = S^1 \times S^1$ 的基本群拆解为两个圆周 $S^1$ 基本群的直积。已知圆周的基本群同构于整数加法群 $\mathbb{Z}$，从而推导出环面的基本群同构于 $\mathbb{Z} \times \mathbb{Z}$。

**证明过程：**

1. **确定空间的乘积结构**： 根据环面（Torus）的定义，空间 $T$ 可以表示为两个单位圆周的笛卡尔积，即 $T = S^1 \times S^1$。设基点为 $t_0 = (1, 1)$，其中 $1 \in S^1$ 是圆周的基点。
    
2. **应用乘积空间基本群定理 (Theorem 3.7)**： Rotman 教材中的**定理 3.7** 指出，对于任意两个带基点的拓扑空间 $(X, x_0)$ 和 $(Y, y_0)$，乘积空间的基本群与其分量空间的基本群之间存在如下自然同构： $$\pi_1(X \times Y, (x_0, y_0)) \cong \pi_1(X, x_0) \times \pi_1(Y, y_0)$$ 该同构由投影映射 $p: S^1 \times S^1 \to S^1$（投影到第一个因子）和 $q: S^1 \times S^1 \to S^1$（投影到第二个因子）诱导的同态给出。
    
3. **引用圆周的基本群结果 (Theorem 3.16)**： 根据**定理 3.16**，单位圆周 $S^1$ 的基本群是一个无限循环群，且存在群同构 $d: \pi_1(S^1, 1) \to \mathbb{Z}$，该同构由道路的“度（Degree）”决定。即： $$\pi_1(S^1, 1) \cong \mathbb{Z}$$
    
4. **代数合成与结论**： 将分量空间的结果代入乘积公式中： $$\pi_1(T, t_0) \cong \pi_1(S^1, 1) \times \pi_1(S^1, 1) \cong \mathbb{Z} \times \mathbb{Z}$$ 因此，环面 $T$ 的基本群是一个秩为 2 的自由阿贝尔群，记作 $\mathbb{Z} \oplus \mathbb{Z}$ 或 $\mathbb{Z}^2$。
    

**几何说明：** 该群的两个生成元分别对应于环面上两条基本的闭曲线：一条是经线循环（绕着中心洞转），另一条是纬线循环（沿着环面长周长转）。由于 $T$ 是一个 **H-空间**（即具有连续乘法运算且有单位元的空间），根据**定理 3.20**，其基本群必然是阿贝尔群，这与 $\mathbb{Z} \times \mathbb{Z}$ 的结果完全吻合。

### 3.17：证明 $S^1$ 不是 $D^2$ 的留缩核（Retract）

**证明过程：**

1. **利用留缩核的定义与性质**： 根据定义，若子空间 $A$ 是空间 $X$ 的留缩核，则存在连续映射（收缩映射） $r: X \to A$ 使得 $r \circ i = 1_A$，其中 $i: A \hookrightarrow X$ 是包含映射。
    
2. **应用基本群函子 $\pi_1$ 的特性**： 由于 $\pi_1$ 是一个从带基点的拓扑空间范畴到群范畴的协变函子，连续映射的复合在基本群层面上诱导同态的复合。若收缩映射存在，则对于基点 $x_0 \in S^1$，对应的诱导同态满足 $r_* \circ i_* = (r \circ i)_* = (1_{S^1})_* = 1_{\pi_1(S^1, x_0)}$。
    
3. **确定相关空间的基本群**： 根据定理 3.16，单位圆周的基本群同构于整数加法群，即 $\pi_1(S^1, x_0) \cong \mathbb{Z}$。而单位圆盘 $D^2$ 是凸集且是可缩的，根据推论 3.12（或 4.25），可缩空间的基本群是平凡的，即 $\pi_1(D^2, x_0) = {1}$。
    
4. **导出代数矛盾**： 将上述群结构代入诱导同态序列中，得到 $\mathbb{Z} \xrightarrow{i_*} {1} \xrightarrow{r_*} \mathbb{Z}$。由于任何经过平凡群 ${1}$ 的同态复合必然将 $\mathbb{Z}$ 中的所有元素映射为 $0$，这与要求复合映射为 $\mathbb{Z}$ 上的恒等映射（即 $1 \mapsto 1$）相矛盾。
    
5. **结论**： 这种代数上的不可能说明最初关于收缩映射 $r$ 存在的假设是错误的，因此 **$S^1$ 不是 $D^2$ 的留缩核**。

### 习题 3.18：证明圆盘上的 Brouwer 不动点定理

**证明过程：**

1. **采用反证法假设**： 假设存在一个连续映射 $f: D^2 \to D^2$，它在整个定义域内都没有不动点，即对于所有的 $x \in D^2$，均满足 **$f(x) \neq x$**。
    
2. **构造几何映射 $g$**： 由于 $f(x)$ 与 $x$ 是两个不同的点，它们可以唯一确定一条从 $f(x)$ 出发经过 $x$ 的射线。定义映射 $g: D^2 \to S^1$，其中 $g(x)$ 是该射线与圆盘边界 $S^1$ 的交点。
    
3. **验证映射的连续性与边界行为**： 由于 $f$ 是连续的且 $f(x)$ 始终不等于 $x$，映射 $g$ 也是连续的。观察 $g$ 在边界 $S^1$ 上的行为：若 $x \in S^1$，由于 $x$ 已经位于边界上，根据射线的构造，该射线与边界的交点必然就是 $x$ 本身，即 **$g(x) = x$ 对于所有 $x \in S^1$ 成立**。
    
4. **识别收缩映射并导出矛盾**： 上述构造表明 $g$ 是一个从 $D^2$ 到其子空间 $S^1$ 的连续映射，且其在 $S^1$ 上的限制是恒等映射。这意味着 $g$ 是一个从 $D^2$ 到 $S^1$ 的 **收缩映射 (Retraction)**。然而，根据习题 3.17 的结论，这样的收缩映射并不存在。
    
5. **结论**： 由于假设“无不动点”导出了“存在收缩映射”这一矛盾结论，原假设必然错误。因此，任何连续映射 $f: D^2 \to D^2$ 必然 **至少存在一个不动点**。

### 习题 3.19：关于圆周闭道路度数的性质证明

**证明过程：**

**(i) 若 $f$ 不是满射，则其度数 $\deg f = 0$**

1. **构造提升映射**：设 $f: I \to S^1$ 是以 1 为基点的闭道路。根据提升引理（Corollary 3.15），存在唯一的连续提升映射 $\tilde{f}: I \to \mathbb{R}$ 满足 $\exp \circ \tilde{f} = f$ 且起始条件 $\tilde{f}(0) = 0$。
2. **度数的定义**：根据定义，闭道路的度数等于其提升映射在终点的值，即 $\deg f = \tilde{f}(1)$。由于 $f$ 是闭道路，该值必为整数 $n \in \mathbb{Z}$。
3. **利用介值定理分析**：假设 $\deg f = n \neq 0$。
    - 若 $n \ge 1$，则提升映射 $\tilde{f}$ 的取值从 $\tilde{f}(0)=0$ 变动到 $\tilde{f}(1)=n$。根据 **介值定理（Intermediate Value Theorem）**，连续函数 $\tilde{f}$ 的像集 $\tilde{f}(I)$ 必然包含整个闭区间 $I$。
    - 若 $n \le -1$，同理可知其像集 $\tilde{f}(I)$ 必然包含闭区间 $[-1, 0]$。
4. **推导满射性矛盾**：
    - 已知指数映射 $\exp: \mathbb{R} \to S^1$ 将任何长度为 1 的实数区间（如 $I$ 或 $[-1, 0]$）都映射到整个单位圆周 $S^1$ 上。
    - 由于 $\tilde{f}(I)$ 包含了这样一个长度为 1 的区间，那么复合映射 $f = \exp \circ \tilde{f}$ 的像集必然涵盖整个 $S^1$。
    - 这意味着 $f$ 必须是满射。
5. **结论**：这与题目中“$f$ 不是满射”的已知条件矛盾。因此，假设不成立，其度数必为 **$\deg f = 0$**。

2. **证明满射性**：

    - 当 $t$ 从 0 变到 $1/2$ 时，参数 $2t$ 已遍历区间 $I$，使得 $f(t)$ 在此期间已经覆盖了整个圆周 $S^1$。
    - 因此，**$f$ 是满射**。

    - 由于 $\tilde{f}(I)$ 包含了这样一个长度为 1 的区间，那么复合映射 $f = \exp \circ \tilde{f}$ 的像集必然涵盖整个 $S^1$。
    - 这意味着 $f$ 必须是满射。
5. **结论**：这与题目中“$f$ 不是满射”的已知条件矛盾。因此，假设不成立，其度数必为 **$\deg f = 0$**。

**(ii) 满射且度数为 0 的示例及证明**

1. **构造示例**：设 $u: I \to S^1$ 为标准绕行圆周一周的闭道路，定义为 $u(t) = \exp(t) = e^{2\pi i t}$。构造复合道路 **$f = u * u^{-1}$**，即先正向绕行圆周一周，再反向绕行回到基点。
2. **证明满射性**：
    - 根据道路连接（Concatenation）的定义，在时间段 $t \in [0, 1/2]$ 内，$f(t) = u(2t) = \exp(2t)$。
    - 当 $t$ 从 0 变到 $1/2$ 时，参数 $2t$ 已遍历区间 $I$，使得 $f(t)$ 在此期间已经覆盖了整个圆周 $S^1$。
    - 因此，**$f$ 是满射**。
3. **证明度数为 0**：
    - 已知 $u$ 的度数 $\deg u = 1$。
    - 其逆道路 $u^{-1}$ 的度数为 $\deg(u^{-1}) = -1$。
    - 根据定理 3.16，连接运算诱导的度数满足加法性质：$\deg(u * u^{-1}) = \deg u + \deg(u^{-1})$。
    - 计算得 **$\deg f = 1 + (-1) = 0$**。
4. **结论**：道路 $u * u^{-1}$ 满足满射且度数为 0 的要求。

### 习题 3.20：关于基本群生成元的证明

**证明过程：**

1. **准备阶段与 Lebesgue 数的应用**： 设 $[f] \in \pi_1(X, x_0)$ 是由连续闭道路 $f: I \to X$ 代表的任一元素，满足 $f(0) = f(1) = x_0$。 考虑 $I$ 的开覆盖 ${f^{-1}(U_j) : j \in J}$。由于单位区间 $I$ 是紧致度量空间，根据 **Lebesgue 数引理**，存在一个正数 $\lambda > 0$，使得 $I$ 中任何直径小于 $\lambda$ 的子区间都包含在**某个** $f^{-1}(U_j)$ 中。
    
2. **区间的剖分与路径的分段**： 取正整数 $n$ 使得 $1/n < \lambda$。将区间 $I$ 等分为 $n$ 个小区间 $I_k = [t_{k-1}, t_k]$，其中 $t_k = k/n$，$k = 0, 1, \dots, n$。 令 $x_k = f(t_k)$。由 $\lambda$ 的性质可知，对于每个 $k = 1, \dots, n$，存在索引 $j_k \in J$ 使得 $f(I_k) \subseteq U_{j_k}$。这意味着分段路径 $f|_{I_k}$ 完全落在 $U_{j_k}$ 中，且其端点 $x_{k-1}, x_k$ 均属于 $U_{j_k}$。
    
3. **构造辅助路径与回路**： 根据已知条件 (i)，基点 $x_0$ 属于所有的 $U_j$。由于每个 $U_{j_k}$ 都是道路连通的，对于每一个分点 $x_k$（$k = 1, \dots, n-1$），我们可以在 $U_{j_k}$ 中选择一条从 $x_0$ 到 $x_k$ 的连续道路 $\alpha_k$。 特别地，定义 $\alpha_0$ 和 $\alpha_n$ 为基点 $x_0$ 处的常数道路，因为 $x_0 = x_n = f(0) = f(1)$。

    - 因此，其同伦类 $[h_k]$ 属于诱导同态的像 $\text{im } i_{j_k*}$，其中 $i_{j_k} : (U_{j_k}, x_0) \hookrightarrow (X, x_0)$ 是包含映射。

5. **合成与结论**： 考虑这些回路类的乘积： $$[h_1][h_2]\dots[h_n] = [\alpha_0 * g_1 * \alpha_1^{-1} * \alpha_1 * g_2 * \alpha_2^{-1} * \dots * \alpha_{n-1} * g_n * \alpha_n^{-1}]$$ 根据道路类的复合性质（逆道路相消），上述乘积简化为： $$[h_1][h_2]\dots[h_n] = [g_1 * g_2 * \dots * g_n]$$ 由于 $\alpha_0$ 和 $\alpha_n$ 是常数道路，且 $g_1 * \dots * g_n$ 恰好是对原闭道路 $f$ 的重新参数化，因此： $$[f] = [h_1][h_2]\dots[h_n]$$ 这证明了 $\pi_1(X, x_0)$ 中的任意元素都可以表示为来自各个子群 $\text{im } i_{j*}$ 的元素的乘积。

### 3.23：证明商群 $G/H$ 是拓扑群

要证明商群 $G/H$（配备商拓扑）是一个拓扑群，我们需要验证两个条件：**群乘法**和**求逆运算**在商拓扑下是连续的。

**证明前提：**

- $G$ 是一个拓扑群，其乘法 $\mu: G \times G \to G$ 和求逆 $i: G \to G$ 均连续。
- $H$ 是 $G$ 的正规子群，因此 $G/H$ 在代数上是一个群。
- $v: G \to G/H$ 是自然商映射（$x \mapsto [x]$），根据商拓扑定义，它是一个**商映射（Identification）**。

---

#### 1. 证明求逆运算 $\bar{i}: G/H \to G/H$ 是连续的

**构造关系：** 考虑商群上的求逆映射 $\bar{i}([x]) = [x^{-1}]$。我们可以得到如下复合关系： $$v \circ i = \bar{i} \circ v$$ 即 $G \xrightarrow{i} G \xrightarrow{v} G/H$ 与 $G \xrightarrow{v} G/H \xrightarrow{\bar{i}} G/H$ 相等。

**证明步骤：**

1. **复合映射连续性**：由于 $i$ 是 $G$ 上的求逆（连续）且 $v$ 是商映射（连续），它们的复合映射 $v \circ i$ 也是连续的。
2. **利用商映射的普遍性质**：根据 **Theorem 1.8** 或 **Corollary 1.9**，对于商映射 $v$，一个诱导映射 $\bar{i}$ 是连续的，**当且仅当**复合映射 $\bar{i} \circ v$ 是连续的。这个定理在证明商映射相关映射的连续性是非常好用。
3. **结论**：既然 $\bar{i} \circ v = v \circ i$ 是连续的，则 **$\bar{i}$ 在 $G/H$ 上连续**。

    - **连续性**：在拓扑群中，乘法、求逆以及常数元素的复合运算都是连续的，因此 $\phi$ 是连续映射。
3. **利用连通性进行论证**：
    
    - 已知 $G$ 是**连通的**。由于 $H$ 是**离散空间**，根据拓扑学性质，从连通空间到离散空间的任何连续映射都必须是**常数映射**。(连通空间的像在离散空间中只能是单点，否则像不连通)
    - 通过代入单位元 $x=1$ 计算常数值：$\phi(1) = 1h1^{-1}h^{-1} = 1$。
    - 因此，对于所有 $x \in G$，恒有 $\phi(x) = 1$。
4. **得出交换性结论**： $\phi(x) = 1 \implies xhx^{-1}h^{-1} = 1 \implies xh = hx$。 这意味着 $H$ 中的每个元素 $h$ 都与 $G$ 中的所有元素 $x$ 交换。因此，$H$ 包含在 $G$ 的中心 $Z(G)$ 内，且 **$H$ 本身必为阿贝尔群**。
    
5. **推论：$\pi_1(G/H, 1)$ 的阿贝尔性质**： 若 $G$ 是单连通的（这意味着 $G$ 也是道路连通且连通的），且 $H$ 是离散闭正规子群：
    
    - 根据覆盖空间理论及习题 3.24 的结论，存在同构 **$\pi_1(G/H, 1) \cong H$**。
    - 由于前文已证 $H$ 是阿贝尔群，因此对应的基本群 **$\pi_1(G/H, 1)$ 也是阿贝尔群**。
    - _另一种视角_：$G/H$ 作为一个商群，本身也是一个拓扑群。根据定理 3.20，任何 **H-空间**（包括所有拓扑群）的基本群必然是阿贝尔群。
